## 📌 Overview
CareerMap AI is an **Agentic AI-powered system** that analyzes resumes, identifies skill gaps, and generates structured career roadmaps aligned with real-world job market requirements.

---

## 🎯 Key Features
✨ Resume Analysis  
📊 Skill Extraction  
⚖️ Skill Gap Detection  
🗺️ Personalized Career Roadmap  
🤖 Multi-Agent AI System  
💻 Interactive Web Application  

---

## 🛠️ Tech Stack
- **Frontend:** Streamlit  
- **Backend:** Python, LangChain, LLM APIs (Groq/OpenAI)  
- **Database:** SQLite, CSV  

---

## ⚙️ Workflow
Resume Upload → Skill Extraction → Market Analysis → Gap Detection → Roadmap Generation
